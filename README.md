# Anomaly Detection Toolkit

A small Python/Jupyter project to **detect anomalies (outliers)** in data using a simple end-to-end workflow:
**exploration → preprocessing → modeling → evaluation → visualization**.

## What’s inside
- Exploratory analysis (EDA) to understand the dataset and spot unusual patterns
- Feature preparation (cleaning / scaling when needed)
- Anomaly detection modeling (unsupervised / semi-supervised approaches depending on the notebook)
- Evaluation and interpretation of detected anomalies
- Plots to visualize anomalies and model behavior
